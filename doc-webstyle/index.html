Creando una ruta

Crear nuestra ruta es bastante facil, solo debemos crear otro objeto del tipo web.route como se muestra en la imagen. Para una ruta que permita el método GET solo debemos agregar web.get() como un objeto de la lista urls[] que ya viene creada en blanco con la creación del proyecto. El primer parámetro obligatorio es el path que tendrá el recuro y como segundo parámetro tambien obligatorio una funcion que maneje la petición. Veamos el ejemplo:

/src/routes.py

Note que el ejemplo de arriba hemos puesto como función manejadora o handler una instancia de controlador y la llamada a su método ( en este caso index). Es necesario importar el controlador que por defecto se encuentra en /controllers carpeta.

Además del verbo GET disponemos de los verbos POST - PUT - DELETE como podemos ver en la siguiente foto. El conjunto http-verbo & path son los que determinan a que método del controlador debemos apuntar.

/src/routes.py

Este comportamiento ya está predefinido y al crear el Recurso por defecto se configura todo como en la foto de arriba. Ésto ya provee la implementacion de CRUD básico. Este comportamiento puede ser modificado reescribiendo el método del controlador que desee.
