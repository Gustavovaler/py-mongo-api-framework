## Requirements
#### Python >= 3.5

`$ pip install aiohttp[speedups]`

`$ pip install pymongo`

## Start

`$ git clone http://github.com/gustavovaler/py-mongo-api-framework.git`

`$ cd py-mongo-api-framework`

`$ python3 server.py`
